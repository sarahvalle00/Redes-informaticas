Redes PAN y LAN

Conoce qué son y cómo se utilizan en la vida diaria

¿Qué es una Red PAN?

Una Red de Área Personal (PAN) conecta dispositivos personales a corta distancia.

Las redes PAN permiten la conexión de dispositivos como teléfonos, laptops, relojes inteligentes y audífonos mediante tecnologías como Bluetooth o USB.

¿Cómo se utilizan?
Se usan para transferir archivos entre dispositivos, conectar accesorios inalámbricos o compartir internet desde un celular.

¿Qué es una Red LAN?

Una Red de Área Local (LAN) conecta dispositivos dentro de un área limitada como una casa o escuela.

Las redes LAN permiten compartir recursos como impresoras, archivos y conexión a internet dentro de un mismo edificio.

¿Cómo se utilizan?
Se utilizan en hogares, oficinas y centros educativos para conectar computadoras mediante cables internet o WiFi.
